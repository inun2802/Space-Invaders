# Space Invaders (Unity)

A retro-style fixed shooter game built in Unity where players fight waves of invading aliens across multiple levels.

## Features
- Multiple difficulty modes (Easy, Hard, Secret)
- Boss fight level
- Sound effects and background music (self created)
- Animated enemies and player projectiles
- Level selection system

## Controls
- Move: Arrow Keys / A-D
- Shoot: Spacebar / Click

## Tech Stack
- Unity (C#)
- Custom game scripts (GameManager, Player, Invaders, etc.)
- Sprite-based animation system
